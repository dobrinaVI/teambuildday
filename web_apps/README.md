This is the Dataiku project webapp sync folder.

- Each webapp lives under `web_apps/<webappId>/` for its code files (for Streamlit, `backend.py`).
- The webapp definition lives in `web_apps/<webappId>.json`.

If you see any other top-level `webapps/` folder, that's **not** a Dataiku project webapp folder (it’s commonly used for plugin webapps or local source copies).
